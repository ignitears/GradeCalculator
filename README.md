# Thai GPA Calculator 🎓

A clean, responsive, and bilingual web application designed to calculate Grade Point Averages (GPA) based on the Thai educational system (4.0 scale). This tool allows students to easily track their grades by entering them manually, pasting tables from Word/Excel, or uploading Excel files directly.

## ✨ Features

* **Bilingual Support:** Instantly toggle between English and ภาษาไทย without reloading the page.
* **Multiple Import Options:**
  * **Upload Excel:** Read `.xlsx` and `.xls` files directly.
  * **Smart Paste:** Copy tables from Excel or Word and paste them in. Opens a review modal before adding data.
  * **Manual Entry:** Add, edit, or delete rows on the fly.
* **Export to Excel:** Download your current subject list and grades back to an `.xlsx` file for safekeeping.
* **Interactive UI:** Smooth animations, mobile-friendly responsive design, and dynamic color-coding (grades automatically highlight green or red).
* **Target GPA Tracker:** Real-time calculation as you type or check/uncheck specific subjects.

## 🚀 How to Use

Since this project uses plain HTML, CSS, and JavaScript, there is no complex installation required.

1. Download the repository files (`index.html`, `style.css`, `script.js`).
2. Put all three files in the same folder.
3. Double-click `index.html` to open it in any modern web browser.

### ⚠️ Important: Excel Upload Format
If you are uploading an Excel file, the data must be on the **first sheet** and follow this exact column structure:
* **Row 1:** Ignored (Reserved for headers like "Subject", "Credits", "Grade")
* **Column A:** Subject Name
* **Column B:** Credits (Numbers only)
* **Column C:** Grade (0.0 to 4.0)

## 🛠️ Technologies Used

* **HTML5 / CSS3:** Custom styling with CSS Grid, Flexbox, and animations.
* **Vanilla JavaScript:** Core logic, DOM manipulation, and language handling.
* **[SheetJS (xlsx)](https://sheetjs.com/):** Used via CDN to parse and export Excel files entirely in the browser.
